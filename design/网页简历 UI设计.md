## 一、视觉风格规范

### 整体视觉调性

**温暖 · 极简 · 干净 · 专业**

以米白和纯白为主色调，构建温和、有呼吸感的视觉基底。通过不同明度的暖白/灰色层次来区分板块，不依赖鲜艳色彩，而是用留白、卡片层次和细腻的阴影来创造节奏感。整体气质偏向"安静的自信"——不张扬，但细节经得起端详。

---

### 配色方案

#### 主色板 —— 暖白 / 米白系

| 色名 | 色值 | 用途 |
|------|------|------|
| 纯白 | `#FFFFFF` | 页面主背景、卡片背景 |
| 米白 | `#FAF4EE` (rgb(250,244,238)) | 交替板块背景（Hero、教育经历），与白色区形成柔和切换 |
| 暖灰-极浅 | `#F5F0EB` | 卡片背景、区块微底色 |
| 暖灰-浅 | `#EBE5DE` | 分割线、装饰线、标签背景 |
| 暖灰-中 | `#C4BAB0` | 占位图标、进度条轨道 |
| 暖灰-深 | `#8C7E72` | 辅助说明文字、次要信息 |

#### 强调色 —— 作为品牌点缀（克制使用）

| 色名 | 色值 | 用途 |
|------|------|------|
| 深灰褐 | `#5A4E44` | 链接色、强调文字、时间线、CTA 按钮边框/文字 |
| 炭黑 | `#2C2826` | 大标题、重要标题文字 |
| 点染色 | `#D4A76A` (暖金) | 极少量点缀——CTA 按钮填充、icon 活跃态 |

> 💡 **配色理念**: 主色域全部在暖白-米白-暖灰之间，用 ONE 个低饱和度暖金 `#D4A76A` 作为唯一的"有彩色"点缀，使用比例控制在 5% 以内。保持整体干净、安静，但有一处微妙的视觉记忆点。

#### 中性色

| 色名 | 色值 | 用途 |
|------|------|------|
| 纯白 | `#FFFFFF` | 卡片、主要背景 |
| 暖灰-标签 | `#F0ECE7` | 技能标签、技术栈标签背景 |
| 暖灰-分割 | `#E0DAD3` | 分割线、边框 |
| 暖灰-文字 | `#6B6158` | 正文辅助色 |
| 正文黑 | `#2C2C2C` | 正文主色 |
| 标题黑 | `#1A1A1A` | 大标题 |

### 字体规范

| 层级 | 字号 | 字重 | 行高 | 使用场景 |
|------|------|------|------|----------|
| **大标题** | 48px / 3rem | Bold / 700 | 1.2 | 姓名 Hero |
| **副标题** | 24px / 1.5rem | SemiBold / 600 | 1.4 | 板块标题、定位标语 |
| **卡片标题** | 20px / 1.25rem | SemiBold / 600 | 1.4 | 作品名称、公司名称 |
| **正文** | 16px / 1rem | Regular / 400 | 1.6 | 描述文字、工作经历详情 |
| **小字** | 14px / 0.875rem | Regular / 400 | 1.5 | 技术栈标签、时间戳、辅助信息 |
| **极小标签** | 12px / 0.75rem | Medium / 500 | 1.4 | 课程名、次要标签 |

**字体选择（按优先级）**

```css
font-family: 'Inter', 'Noto Sans SC', 'PingFang SC', 'Microsoft YaHei', system-ui, sans-serif;
```

- Inter 作为英文字体主力，几何感干净
- Noto Sans SC 作为中文字体备选（与Inter搭配协调）
- 代码/技术栈标签可用等宽：`'JetBrains Mono', 'Fira Code', monospace`

| 层级 | 值 | 使用场景 |
|------|-----|----------|
| 大圆角 | 16px / 1rem | 大卡片、头像 |
| 中圆角 | 12px / 0.75rem | 普通卡片、截图容器 |

### 阴影规范

| 层级 | 值 | 使用场景 |
|------|-----|----------|
| 微阴影 | `0 2px 8px rgba(0,0,0,0.04)` | 普通卡片默认态 |
| 中阴影 | `0 8px 24px rgba(0,0,0,0.06)` | 卡片悬浮(hover)态、项目截图 |
| 大阴影 | `0 16px 48px rgba(0,0,0,0.08)` | Hero 头像 |

> 阴影色调偏暖：`rgba(90, 78, 68, 0.06)` 比纯黑阴影更协调。

### 卡片样式

```
┌─────────────────────────────┐
│  ┌──────┐                   │
│  │ 截图  │   标题 / 描述...  │
│  └──────┘                   │
│  标签 [标签1] [标签2]       │
│  🔗 链接                    │
└─────────────────────────────┘
```

- **背景**: 纯白 `#FFFFFF`
- **边框**: 无边框，或 1px `#E0DAD3` 极淡描边
- **内边距**: 24px / 1.5rem
- **间距**: 卡片之间 24px / 1.5rem
- **悬浮效果**: 中阴影升起 + `transform: translateY(-2px)` + 过渡 0.25s ease

---

### 图标风格

- 使用 [Lucide](https://lucide.dev/) 线条风格图标
- 尺寸：24px 常规 / 20px 内联 / 48px Hero 社交图标
- 描边宽度：1.5px
- 颜色跟随 surrounding 文字色，hover 时变为暖金 `#D4A76A`

## 附录：页面间滚动与过渡

### 全局过渡效果

```
板块滚动进入视口 → IntersectionObserver 触发
    → opacity: 0 → 1
    → transform: translateY(30px) → translateY(0)
```
